# vue-order-lunch
vue.js + vuex + webpack 

# 进入文件夹
cd vue-order-lunch

# 安装依赖
npm install 或 yarn(推荐)

# 开启本地服务器localhost:8080
npm run dev

# 发布环境
npm run build
